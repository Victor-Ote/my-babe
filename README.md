# ❤️ Para Você Fer

Uma página romântica desenvolvida em HTML, CSS e JavaScript puro, criada para exibir fotos especiais de forma dinâmica e interativa.

## Funcionalidades

* Exibição de fotos em formato circular.
* Troca automática e aleatória de imagens com efeito de fade.
* Fundo animado com frases de amor em diversos idiomas.
* Botão interativo que exibe mensagens e lembranças aleatórias.
* Layout responsivo para computadores, tablets e celulares.
* Animações suaves para tornar a experiência mais agradável.

## Estrutura do Projeto

```text
/
├── index.html
└── media/
    ├── image1.jpeg
    ├── image2.jpeg
    ├── image3.jpeg
    └── ...
```

## Como Utilizar

1. Coloque todas as fotos na pasta `media`.
2. Atualize a lista de imagens no arquivo `index.html`, se necessário.
3. Abra o arquivo `index.html` em qualquer navegador moderno.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript Vanilla

## Personalização

Você pode alterar:

* As fotos exibidas.
* As frases animadas no fundo.
* As memórias exibidas ao clicar no botão.
* Cores, animações e estilos visuais.
